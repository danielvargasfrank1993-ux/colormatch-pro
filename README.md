# 🎨 ColorMatch Pro — Sistema Inteligente de Formulação e Colorimetria Industrial

O *ColorMatch Pro* é uma solução completa desenvolvida para indústrias de tintas e revestimentos que buscam modernizar o controle de qualidade, acelerar o acerto de cores e eliminar o desperdício de pigmentos na produção.

---

## 🚀 Principais Funcionalidades

- *📸 Leitura por Câmera em Tempo Real:* Captura a cor da amostra diretamente pela câmera do celular (iOS/Android) ou computador.
- *🧮 Cálculo Automático de Batelada:* Algoritmo tristímulo (CIELAB) para dosagem exata de pigmentos (CB, Y25, 930, SH) por caixa ou balde.
- *📄 Geração de Ordem de Produção em PDF:* Emissão rápida de ordens de pesagem para o chão de fábrica.
- *🏢 Módulo Corporativo Multi-empresa:* Banco de dados isolado onde cada cliente gerencia e salva seu catálogo exclusivo de cores.
- *📱 Responsivo e Acessível:* Funciona direto no navegador (Safari, Chrome) sem necessidade de instalação pesada.

---

## 🛠️ Tecnologias Utilizadas

- *Linguagem:* Python
- *Interface Web:* Streamlit
- *Processamento de Imagem:* Pillow (PIL)
- *Geração de Documentos:* ReportLab
- *Banco de Dados:* SQLite3

---

## 🏭 Aplicação Industrial

Ideal para laboratórios de desenvolvimento, coloristas, controle de qualidade e gestão de chão de fábrica em indústrias de tintas imobiliárias, automotivas e industriais.
